# 숫자 야구 게임

## 기능 목락
- [ ] 1부터 9까지의 서로 다른 임의의 수 3개를 생성한다.
- [ ] 컴퓨터의 수(3자리)와 플레엉의 수 (3자리)를 비교할 수 있다.
    - [ ] 몇 개의 숫자가 같은지를 알 수 있다.
    - [ ] 같은 수가 다른 자리에 있으면 볼이다.
    - [ ] 같은 수가 같은 자리에 있으면 스트라이크이다.
    - [ ] 같은 수가 전혀 없으면 낫싱이다.
    
## 기능 요구 사항
기본적으로 1부터 9까지 서로 다른 수로 이루어진 3자리의 수를 맞추는 게임이다.
